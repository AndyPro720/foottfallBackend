# Plan 05-02: Role-Based Access Control (RBAC)

Ensure users only see their own properties unless they are admins, and implement a `users` collection for role management.

## Tasks

- [ ] Create `src/backend/userRoleService.js`:
    - Function to get or create a user document in `/users` on first sign-up
    - Default status: `agent`
- [ ] Modify `src/backend/inventoryService.js`:
    - Add `createdBy` field to every property on creation (with `auth.currentUser.uid`)
    - Wrap `getProperties` to add a `where("createdBy", "==", auth.currentUser.uid)` filter
    - Handle visibility override for `admin` role by checking user document
- [ ] Update `firestore.rules`:
    - Add rule to allow read/write ONLY if `resource.data.createdBy == request.auth.uid` or user has `admin` role
    - Secure the `/users` collection: users can only read/edit their own profile
