# Plan 2.1: Data Access Layer (Firebase Wrapping)

## Objective
Build out the data access layer that wraps the Firebase Firestore and Storage SDKs to provide clean, reusable CRUD functions for the `inventory` collection. This fulfills the "Core API" requirement without a dedicated node backend.

## Context
- .gsd/ROADMAP.md
- .gsd/phases/1/SCHEMA.md
- src/backend/firebaseConfig.js
