# Plan 07-04: Premium UI Polish

## Objective
Elevate the visual aesthetic to a premium, Linear/Stripe-inspired SaaS look. Increase depth, improve typography hierarchy, and enhance interactive states.

## Tasks

<task id="1">
<title>Enhance card depth, borders, and gradients</title>
<read_first>
- src/style.css (lines 6-60: variables, lines 205-240: cards)
</read_first>
<action>
In `src/style.css`:

1. Update css root variables for more premium colors and shadows:
   - Add a subtle background gradient to `.card` elements (e.g., `background: linear-gradient(180deg, rgba(255,255,255,0.03) 0%, rgba(255,255,255,0) 100%), var(--bg-raised);`)
   - Increase shadow depth on `.card-interactive:hover` (e.g., `box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);`)
   
2. Improve typography scale (lines 163-195):
   - Make `.text-display` adapt better on mobile (line 743). Change to `font-size: clamp(24px, 5vw, 32px);` to avoid sharp breakpoint switches.
   - Adjust letter spacing slightly on text-body for better readability (`letter-spacing: -0.01em;`).

3. Enhance buttons (lines 278-337):
   - Add a subtle inner box-shadow to `.btn-primary` to give it a 3D pill feel (`box-shadow: inset 0 1px 1px rgba(255, 255, 255, 0.2);`).

4. Increase file upload affordance (lines 489-502):
   - Change `.file-upload-zone` background to `rgba(255, 255, 255, 0.02)`.
   - On hover, make border solid instead of dashed, or increase brightness.

5. Adjust thumbnail sizes:
   - Increase `.card-thumbnail-wrapper` from 56px to 72px (lines 234-237).
</action>
<acceptance_criteria>
- `.card` CSS contains `linear-gradient`
- `.text-display` CSS contains `clamp(`
- `.btn-primary` CSS contains `box-shadow: inset`
- `.card-thumbnail-wrapper` CSS width and height are `72px`
</acceptance_criteria>
</task>
