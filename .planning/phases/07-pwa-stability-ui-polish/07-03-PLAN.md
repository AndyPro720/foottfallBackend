# Plan 07-03: Form Configuration & Required Fields

## Objective
Simplify property registration by making all fields optional except the Property Name. Default all sections to collapsed (except Property Information) to reduce mobile UI clutter.

## Tasks

<task id="1">
<title>Make fields optional and ensure sections are collapsed</title>
<read_first>
- src/config/propertyFields.js
</read_first>
<action>
In `src/config/propertyFields.js`:

1. Search for `required: true` across the entire document.
2. Keep `required: true` ONLY for the `name` field in the `propertyInfo` section.
3. Remove `required: true` (or change to `required: false`) from:
   - `buildingType`
   - `location`
   - `contactName`
   - `contactInfo`
   - `size`
   - `price`
4. Verify the `collapsed` properties on the section definitions at the bottom of the file (around line 60):
   - `propertyInfo` should have `collapsed: false`
   - `contact` should have `collapsed: true`
   - `unitSpecs` should have `collapsed: true`
   - `facilities` should have `collapsed: true`
   - `photos` should have `collapsed: true`
</action>

</task>
